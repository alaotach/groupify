# Groupify - Group your tabs

>  A friend of mine was working as an open source contributor for some organizations during summer of code programs happening these days and he had way too many tabs open which he couldn't manage to organize and he didn't want the clutter of multiple windows eating ram on his machine so he asked me to build a chrome extension to bundle the tabs into saved groups.

## What is Groupify?
Groupify is a chrome extension that lets users bundle their open tabs into groups.

## Features

1. one click save entire window, so your entire window becomes a group.
2. / to search in saved groups
3. right click any tab and shove it into any group

## Shortcuts:
- open extension → alt + shift + o
- group selected tabs → alt + shift + g
- save current window → alt + shift + s
- focus search → just hit /
- new group → start typing
